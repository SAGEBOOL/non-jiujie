Realm 01 · 九界第一界

仙界

天上仙城 · 大道至高
仙界是修士飞升后的归宿。以道为尊，法则至上。表面祥和神圣，内里却暗流涌动——仙帝统御、仙域争霸、资源争夺从未停止。

# 九界第一层 # 大罗金仙上限 # 九大仙域 # 灵气充裕 # 飞升通道
Basic Geography · 基本地理

地理信息总览

仙界基础设定

  • 整体形态：一片悬浮于虚空的大陆群，由九大仙域组成，每域以一座仙山为核心。天空呈淡金色，仙气缭绕，常有仙鹤、神兽飞掠。
  • 面积：约为人界（天元界）的十倍。分为九大仙域：中央仙域（仙帝直辖）、东极仙域、南极仙域、西极仙域、北极仙域、东南仙域、西南仙域、东北仙域、西北仙域。
  • 气候环境：四季如春，仙气浓郁，灵气充盈。对凡人而言是仙境，对仙人而言是日常。
  • 对人族修士的影响：仙气可大幅提升修炼速度，但长期停留会使凡体仙化，无法再回人界。
Landmarks · 重要地标

仙界地标

Palace
仙帝宫
仙帝居所，中央仙域核心。由万年仙玉铸成，仙气氤氲。仙界权力中心，裁决一切仙域纷争。
Mountains
九大仙山
各仙域核心，飞升者第一站。每座仙山都是一处洞天福地，蕴藏无尽机缘。
Market
仙市
仙界最大交易场所，位于中央仙域。各仙域修士在此交换丹药、法宝、功法。
Pool
飞升池
飞升者初入仙界稳固仙体之地。在此淬炼凡躯，适应仙界灵气，否则易被仙气反噬。
Gate
仙门
与人界连接的飞升通道入口。每千年开启一次，接纳人界修士飞升仙界。
Prison
仙狱
关押犯禁仙人的禁地。仙界最黑暗之处，囚禁着触犯仙界铁律的仙人。
History · 历史渊源

仙界编年史

太古时期
初代仙帝开辟仙界 初代仙帝开辟仙界，建立九大仙域，制定仙界秩序。奠定仙界万年基业。
上古时期
仙魔之战 仙界与魔界大战，仙帝重伤，仙界元气大伤。此后仙界与魔界签订互不侵犯条约。
中古时期
飞升通道稳定 仙界与人界飞升通道稳定，每千年有数十位修士飞升。仙界逐渐壮大。
近古时期
仙域争霸 九大仙域争夺资源，爆发仙域大战。仙帝出手镇压，重新划定势力范围。
当代
暗流涌动 仙界与圣界、妖界保持有限往来，警惕魔界渗透。内部问题：仙帝寿元将尽，继承人之争暗流涌动。

仙帝传承制度

  • 传承方式：仙帝非世袭，由上一任仙帝指定或"仙界大会"推举。
  • 历任仙帝：历代仙帝多为大罗金仙巅峰，寿元不可考。
  • 仙帝职能：统领仙界，裁决仙域纷争，守护飞升通道，抵御外敌入侵。
  • 仙庭代管：当代仙帝昊天性恪公正威严，极少理事，日常由"仙庭"代管。
Cultivation System · 修行体系

仙界修行层级

Level 01
天仙
仙界最低层级，多为飞升初期修士。仙力微弱，需苦修晋升。
Level 02
真仙
仙力初步凝实，可独当一面。仙界基层主力。
Level 03
玄仙
仙力大成，可修炼高阶仙法。仙界中层骨干。
Level 04
金仙
触碰法则之力，寿命近乎无限。一仙域之主。
Level 05
大罗金仙
超脱时空，万劫不灭。仙界顶层战力，历代仙帝之境。
Level 06
仙帝
大道至尊，执掌仙界。唯一存在，诸天万界之巅。
Core Rules · 核心规则

仙界铁律

  • 以道为尊：仙界一切以大道法则为最高准则，实力决定地位。
  • 法则至上：掌握越多法则，实力越强。仙帝掌握大道本源。
  • 资源争夺：仙丹、法宝、洞天福地等修炼资源极为有限，竞争激烈。
  • 飞升限制：每千年飞升通道开启，飞升人数有限制，竞争激烈。
  • 仙狱铁律：触犯仙界铁律者，打入仙狱，永世不得超生。
  • 仙域边界：各仙域有明确边界，越界需经对方仙君同意，否则视为挑衅。
